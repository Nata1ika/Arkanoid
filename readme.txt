Сделать простой арканоид. Есть поле с блоками, есть управляемая игроком бита, есть шарик
что отражается от стен, блоков, биты. Блоки разбиваются от столкновений шариками. Шарик
считается потерянным если он улетел за нижнюю часть экрана. Игрок считается выигравшим
если он уничтожил все блоки, проигравшим если лишился всех шаров. При выигрыше игрок
получает сообщение что он победил с кнопкой перезапуска уровня. При проигрыше игрок
получает сообщение что он проиграл и кнопку для запуска игры заново.
Графика — простых цветные прямоугольников и кругов будет более чем достаточно. Звук не
нужен.
Желательно, но не обязательно:
• Подсчёт очков.
• Более сложные блоки.
• Бонусы.
• Жизни.
Обязательные условия:
• Сделано должно быть на Unity и C#, без использования каких либо сторонних
библиотек или ассетов.
• Версия Unity любая.
• Должен быть предоставлен полный проект (папки Assets и ProjectSettings).
